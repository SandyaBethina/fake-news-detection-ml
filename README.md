# fake-news-detection-ml
An end-to-end Machine Learning project for detecting fake news articles using Natural Language Processing (NLP). The project includes text preprocessing, feature extraction using TF-IDF, multiple classification models, model evaluation, and prediction of real vs. fake news articles.
# Features
Data Cleaning
Text Preprocessing
Stopword Removal
Stemming / Lemmatization
TF-IDF Vectorization
Model Training
Model Evaluation
Confusion Matrix
Accuracy Comparison
Predict New News Articles
